<h2>控制篮球机器人弹射程序</h2>
从其他节点接收弹射指令，转化为下位机协议后控制篮球机器人弹射装置
<h4>控制方式如下</h4>
<ul>
	<li>id = 1 ----------- 弹射装置弹出</li>
</ul>

<h5><font color="red" size="20px">warn:</font></h5>
<p>为确保在执行下一个指令时，弹射动作已完全完毕，故添加了sleep，确保球弹出后再移动</p>
